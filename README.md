# ManifestoChallenge
Escolhi o jogo [Micro-Games: Platformer](https://assetstore.unity.com/packages/templates/micro-games-platformer-151055) para fazer a modificação da atividade proposta. Escolhi esse jogo por ter mais afinidade com jogos 2D. As modificações feitas foram coisas que identifiquei que estavam faltando para o template de um jogo de plataforma. Abaixo eu mostro quais as modificações foram feitas através dos commits feitos nesse repositório. Além disso, também listo os códigos que modifiquei e os que criei.
